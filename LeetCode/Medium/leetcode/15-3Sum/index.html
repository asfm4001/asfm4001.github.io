發表於 2026-03-05 更新於 2026-03-06 分類於 LeetCode ， Medium 閱讀次數： 文章字數： 1.6k

種類 : two pointers, hash table
難度 : medium
題目 : 15. 3Sum

思維邏輯

  1. 區分正、負、零組合
    1. 篩選至list，區分正數、負數、零
    2. 建立正負set
    3. 若有0，檢核正負是否有對應值
    4. 檢核是否有[0, 0, 0]
    5. 負數任2組合，檢核正數set是否相符
    6. 正數任2組合，檢核負數set是否相符

解法

時間複雜度: O(n+2n2) -> O(n2)
空間複雜度: O(2n)

難點

  1. 三個元素不重複，但輸出時需過濾掉重複值
  2. 無排序(有排序使用雙指針較有效率)
