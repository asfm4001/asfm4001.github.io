203. Remove Linked List Elements

發表於 2025-04-25 分類於 LeetCode ， Easy 閱讀次數： 文章字數： 1.3k

種類 : linked list
難度 : easy
題目 : 203. Remove Linked List Elements

思維邏輯

  1. 歷遍list
    1. 若cur_node.data == target:
      1. 若cur_node為head:
        更新head
      2. 若cur_node為尾(next==None):
        更新last.next, 並更新cur_node為last_node
      3. 否則:
        更新last.next為cur_node.next
    2. 紀錄last_node
    3. 更新cur_node
  2. 返回head

解法

時間複雜度: O(n)
空間複雜度: O(1)

難點

  1. 當target重複出現時，需更新cur_node == last_node

最佳解法

時間複雜度: O(n)
新增一個node在head前，就無須紀錄last_node
結束必須返回此node.next作為新的head
