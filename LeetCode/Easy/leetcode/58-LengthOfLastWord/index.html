58. Length Of Last Word

發表於 2025-04-18 更新於 2025-04-25 分類於 LeetCode ， Easy 閱讀次數： 文章字數： 918

種類 : list
難度 : easy
題目 : 58.Length of Last Word

思維邏輯

  1. 由字串末端向前推進，使用two point
  2. right為字串尾，逐一向左，直至 s[right] != ' '
  3. left隨right向左
  4. 當 s[right] != ' ' 時，left向左，直至left == 0 or s[left] != ' '

解法

時間複雜度: O(n)
空間複雜度: O(1)

難點

  1. two point範圍容易搞混，right必須歷遍[0, n]，故為len(s) - 1
  2. left需在字串開頭([0])與值為空的時候停止計算
  3. 避免在迴圈中重複計算單字的長度

最佳解法
